## Overlayroot File-System for Jetson
Tested on:
- AGX Xavier - JetPack-4.6


### Setup
```
$ sudo apt update
$ sudo ./setup.sh
```

### Enabling the Overlayroot File-System
```
$ sudo ./enable_overlayroot.sh
$ reboot
```

### Disabling the Overlayroot File-System
```
$ sudo ./disable_overlayroot.sh
$ reboot
```
